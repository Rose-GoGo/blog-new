这是一个使用creact-react-app搭建的管理项目.因为项目使用了eject，所以跟使用webpack直接搭建的并没有多大区别，项目计划完成权限管理，图表展示等功能。


## 文档结构

After creation, your project should look like this:

```
my-app/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    App.css
    App.js
    App.test.js
    index.css
    index.js
    logo.svg
